
#  Homedo – Household Services Management System

A web-based platform for booking household services like cleaning, plumbing, electrician work, and more.  
Built using **Django (Python)** with role-based dashboards for users, service providers, and admin.

---

##  Features

###  Customer Features
- User registration & login system
- Book household services easily
- Track booking status (Pending / Accepted / Declined)
- Product ordering system
- Payment integration (Razorpay / COD)
- View booking history

###  Service Provider Features
- View incoming service requests
- Accept or decline bookings
- Manage assigned services

###  Admin Features
- Manage users and service providers
- Add / update / delete services
- Monitor bookings and payments
- Full system control dashboard

---

##  Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Database:** SQLite / MySQL
- **Payment Gateway:** Razorpay
- **Tools:** VS Code, Git

##  Installation

```bash
git clone https://github.com/your-username/homedo.git
cd homedo

python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate

pip install -r requirements.txt

python manage.py makemigrations
python manage.py migrate

python manage.py createsuperuser

python manage.py runserver
````
